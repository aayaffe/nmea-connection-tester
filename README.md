# NMEA Connection Tester

Portable handheld device for testing, validating, and troubleshooting NMEA 0183 and NMEA 2000 marine electronic connections.

Repository structure:

```
nmea-connection-tester/
├── hardware/
│   ├── schematics/
│   ├── pcb/
│   ├── bom/
│   └── datasheets/
├── firmware/
│   ├── src/
│   ├── lib/
│   ├── test/
│   └── docs/
├── enclosure/
│   ├── cad/
│   └── 3d-print/
├── docs/
│   ├── user-manual/
│   ├── assembly/
│   └── api/
└── tools/
```

See /docs and each subfolder for more details.
